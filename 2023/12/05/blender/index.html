blender实用技巧和典型案例

Blender学习中的记录。不定期更新。

基本建模技巧

Tips

大部分的指令不能通过 ESC 退出，即使 ESC 了也会起作用，比如： - 环切的时候，esc 表示在中间环切。 - extrude 的时候，esc 表示创建新边，但是不挤出。这时候进行缩放会发现是原地缩放。 - Shift+C，让 3D 游标回到世界中心。 - ctrl+A，选择应用所有的变换。变换进行了之后，会在变换界面显示出来。如果使用阵列等命令，会在变换之前的基础上进行操作。所以为了防止原形毕露，需要应用变换，使得新的样式变成基本的样式。 - H 隐藏选中的物体。Alt+H 取消全部隐藏。 - F 可以连接选中的 primitives。点和线都可以。

开口封口

内容较为简单，直接叙述。 在选中需要封口的一个环形的边之后，使用快捷键 F 就可以封口。

如果直接按 F 的时候，产生的是黑色的面，证明你选择边的顺序不对，生成了重叠的面。去掉部分的边，然后多次按 F 也能达到最佳的效果。

快速选中循环边

使用 Option/Alt + LMB 可以快速选中一条边的循环边。

平面开圆

首先通过调整吸附模式，通过拖动 3D 游标让它吸附到你需要的地方的中心点（圆心）。

删掉中间的面。

然后在中间 shift+A，添加一个圆。调整使之符合要求。

选中两条循环边之后，右键合并循环边。注意边是有顺序的，如果顺序不对，会发现中间被填上了。这样是错误的。

然后 extrude 即可。任务完成。

旋转阵列

两种方法，第一种是使用编辑模式下的 Spin 工具，在 3D 游标处展开旋转轴。而且可以自动 merge。

第二种办法是使用阵列修改器。好处是可以实时查看。坏处是操作繁琐。 在进行阵列修改之前，先在需要的原点处设置一个空物体，然后将空物体设置成阵列修改的依据，对空物体进行操作即可。

两种办法都会出现的问题是，物体的 origin 可能和 3D 游标不是你想象的那样和谐。需要在 Object 模式下，右键重设 Origin。

Shift+S

Shift+S 可以调出 cursor 的移动面板，方便的选择 Cursor 的位置。特别地，可以让 Cursor 移动到面的中心，不用像之前一样手动划分了。

模拟小键盘

在没有小键盘的情况下，参考手册 Input — Blender Manual 可以调节出来模拟小键盘的功能。通常不使用。

间隔选中

菜单里面，select->checker deselect 可以隔一个取消选中一个。

合并点到中心

这种情况属于 " 合并 "，使用快捷键 m 调出合并菜单，合并到中心即可。

平滑着色

右键可以进行平滑着色。

如果着色不对，说明法向量没有正确的计算，需要在侧变量手动勾选法向的自动平滑。

点的基本操作

【5分钟Blender小技巧】-编辑点_哔哩哔哩_bilibili 这个视频教了很多。

点的移动

GG，按照边移动。 ctrl+shift+B，能够对点倒角。ctrl+B，只能对边倒角。 J 连接两个点。

横切面

在一条边上选中若干个点，按 V 可以切开

按 alt+V 可以切开的同时在中间连接面。

点的平滑

选中多个点，然后右键，选择平滑点

然后可以 shift+R 重复这一操作直到满足要求。

点的对齐

点吸附模式下，即使仅在一个轴上运动，也能进行对齐。

如果有多个点需要对齐，可以考虑使用缩放模式的缩放功能。原理是这样的：

选中多个点，让它们在某一个轴上缩放，这时候如果缩放倍数为 0，那么它们在这个轴上的距离就都是零了，也就意味着在同一条直线上。

这时候就相当于对齐了。好妙的思路。

顶点组

ctrl+G，定义新的顶点组。可以用来快速选择，在顶点组界面进行 select 就直接选中了。 还有其他功能，刷参数之类的。

球化点

alt+shift+S，可以让选中的所有点球化分布。

基本操作误解

挤出选区 Extrude

挤出 Extrude 有五种不同的操作，即 - Extrude Region - Extrude Manifold - Extrude Along Normals - Extrude Individuals - Extrude to Cursor 具体细节参考 Extrude — Blender Manual，这里给出大致的理解： - 挤出选择的区域 - 挤出选择的区域，并且作为 Manifold 处理，冲突的边和面都会被转化，使之符合 Manifold 定义。 - 各选区按照自己的法线方向挤出，而不是一个统一的方向。 - 各选区挤出之后，各自获得单独的中心点，因此之后按照中心点操作的时候，各面是独立的。也就是说，相接触的边不会被合并成一条边。 - 直接挤出到鼠标点击的位置。
